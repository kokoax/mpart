# MpdClient
[![Travis](https://api.travis-ci.org/kokoax/mpdart.svg)](https://travis-ci.org/kokoax/mpdart)

Must write
``` config/config.exs
use Mix.Config
config :mpd_client,
  api_token: "Last.fm API Token",
  musicdir: "/path/to/your/Musicdir",
  redis_host: "localhost",
  redis_port: 6379
```

## Dependencies
- MPD
- Redis
